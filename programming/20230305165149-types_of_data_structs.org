* Linear Data Structures
A Linear data struct is one that is arranged sequentially in memory
They also have only one predeccesor and one successor
Examples:
+ Stack
+ Array
+ Queue
+ Linked list
* None Linear Data Structures
A non linear data struct is one that is not arrange in a sequential order.
Examples:
+ Tree
+ Graph
* Static Data Structs
memory is allocated at compile time
Max size is always fixed

Examples:
+ Array


* Dynamic data structs

memory is allocated at run time, it is flexible

Example:
+ Linked list
