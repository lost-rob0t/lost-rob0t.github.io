directory-local variables

In emacs there is the concept of directory local variables. They get activated when you enter the directory. They are useful for large projects.

Simply put a file named .dir-locals.el, or .dir-locals-1.el if the former is already taken.

M-x add-dir-local-variable can also be used to create one. They will be loaded when emacs enters the directory or any subdirectory containing that file.

direnv is the shell based version of this, and you can use it in emacs too https://github.com/purcell/envrc.
