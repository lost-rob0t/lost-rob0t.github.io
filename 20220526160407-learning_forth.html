Chapter 1

Good organization is:

  • 1. define tasks and give it a name
  • 2. Group related tasks and give the group a name, ect

Forth allows you to structure your code in this manner and does not get in the way.

The Colon is the start of a new definition. words are seperated by whitespace

Fourth is interactive

you can use it just like a repl and type the name of the word adn it will execute.

if you have gforth installed, open a repl by entering "gforth"

Press return and it will return with 'ok'

Enter this 

15 SPACES

When you pressed the enter key forth returned you with 15 spaces

Now lets emit a * 

42 EMIT

This returns a *. 42 is the char code for *
